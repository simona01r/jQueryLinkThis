# LinkThis jQuery plugin
**LinkThis** is a simple jQuery plugin to attach links to elements

## Version
0.2.0

## Usage
    $("span").linkThis({
        href :"http://test.com",
        text :"Click this!",
        target : "_blank"
     });

## Vendors
jQuery - [http://jquery.com](http://jquery.com)
